# ✨ Overview

Prism is a modern desktop application that helps users build custom PCs with ease.

Instead of manually checking compatibility across multiple websites, Prism aims to bring everything together into one elegant application with a clean interface, smart component management, and powerful build tools.

Whether you're a beginner building your first PC or an enthusiast planning a high-end workstation, Prism is designed to simplify the process.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Nihan-11/Prism.git
```

Move into the project

```bash
cd Prism
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python login.py
```
